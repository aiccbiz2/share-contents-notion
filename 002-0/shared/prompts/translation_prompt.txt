# 작업
다음 YouTube 자막을 한국어로 자연스럽게 번역하고 정제해주세요.

# 원본 언어
{source_lang}

# 원본 텍스트
{text}

# 번역 규칙
1. 자연스러운 한국어 표현 사용 (직역보다는 의역)
2. 불필요한 반복 제거
3. 문장 구조 개선 (자막 특유의 끊김 개선)
4. 전문 용어는 영문 병기 (예: 머신러닝 (Machine Learning))
5. 구어체를 자연스러운 문어체로 변환
6. 타임스탬프 정보는 유지

# 가장 중요한 규칙 (필수)
- 원본 텍스트의 각 자막 항목은 "---" 구분자로 분리되어 있습니다
- 번역 결과에서도 반드시 "---" 구분자를 그대로 유지해야 합니다
- 입력에 N개의 "---" 구분자가 있으면, 출력에도 정확히 N개의 "---" 구분자가 있어야 합니다
- 절대로 여러 항목을 합치거나 구분자를 제거하지 마세요

# 출력 형식
번역된 텍스트만 반환해주세요 (JSON이나 다른 포맷 없이).
각 자막 항목 사이의 "---" 구분자를 반드시 유지하세요.
