# YouTube Summary Agent - 프론트엔드 디자인 명세서

## 프로젝트 개요
YouTube 영상의 자막을 추출하고 AI로 요약하여 Notion에 저장하는 웹 애플리케이션

## 기술 스택
- React + TypeScript
- Vite (빌드 도구)
- TailwindCSS (스타일링)
- React Query (서버 상태 관리)

---

## 페이지 구조

### 1. 헤더 (Header)
- 앱 로고/타이틀: "YouTube Summary Agent"
- 부제: "AI 기반 영상 요약 도구"

### 2. 메인 영역 (Main Content)
최대 너비 1152px (max-w-6xl), 중앙 정렬

---

## 주요 컴포넌트 및 기능

### 2.1 URL 입력 섹션 (VideoInput)
**목적**: YouTube URL 입력 및 처리 시작

**UI 요소**:
- 텍스트 입력 필드 (placeholder: "YouTube URL을 입력하세요")
- 3개 버튼:
  - "미리보기" 버튼 - 영상 정보만 먼저 확인
  - "요약 시작" 버튼 (Primary, 강조색) - 전체 처리 시작
  - "초기화" 버튼 - 결과가 있을 때만 표시, 모든 상태 리셋

**상태**:
- 로딩 중일 때 버튼 비활성화 + 스피너 표시

---

### 2.2 요약 내역 섹션 (HistorySection)
**목적**: 최근 30일간 요약했던 영상 목록 보기

**UI 요소**:
- 접이식(Collapsible) 토글 패널
- 토글 버튼: 시계 아이콘 + "요약 내역" 텍스트 + 개수 배지
- 펼쳤을 때:
  - 스크롤 가능한 목록 (최대 높이 400px)
  - 각 항목:
    - 썸네일 이미지 (96px 너비, 16:9 비율)
    - 제목 (2줄 제한)
    - 채널명 | 영상 길이
    - 요약 날짜 | 처리 시간 | Notion 저장 여부 체크 표시

**동작**:
- 항목 클릭 시 해당 요약 결과를 메인 화면에 로드
- 로딩 중 스피너 표시

---

### 2.3 에러 메시지 (ErrorMessage)
**목적**: 오류 발생 시 사용자에게 알림

**UI 요소**:
- 빨간색 배경의 알림 박스
- 경고 아이콘 + 에러 메시지 텍스트
- X 버튼으로 닫기 가능

---

### 2.4 비디오 정보 카드 (VideoInfoCard)
**목적**: 선택된 영상의 기본 정보 표시

**UI 요소**:
- 썸네일 이미지 (320px 너비)
- 영상 제목
- 채널명
- 영상 길이 (시:분:초 형식)
- Notion 링크 (결과 있을 때):
  - "Notion에서 보기" 버튼 (검은색, Notion 아이콘)
  - "URL 복사" 버튼 (회색 테두리)
    - 복사 완료 시 초록색으로 변경 + "복사됨!" 텍스트

---

### 2.5 로딩 상태 (LoadingState)
**목적**: 처리 중임을 시각적으로 표시

**UI 요소**:
- 스피너 애니메이션
- "영상을 분석하고 요약하는 중입니다..." 메시지
- 실시간 경과 시간 표시 (0분 0초부터 카운트)

---

### 2.6 처리 완료 배너
**목적**: 요약 완료 알림 + 소요 시간 표시

**UI 요소**:
- 초록색 배경 박스
- 체크 아이콘
- "요약 완료! 총 소요 시간: X분 Y초"

---

### 2.7 요약 섹션 (SummarySection)
**목적**: AI가 생성한 구조화된 요약 표시

**UI 요소**:
- 섹션 제목: "요약"
- 구조화된 요약 내용:
  - **서론 (Introduction)**: 영상의 전반적 개요
  - **핵심 논점 (Core Topics)**: 주요 주제별 요약
    - 각 논점: 제목 + 상세 요약
  - **핵심 주장/인사이트 (Key Arguments)**: 주요 포인트 목록
  - **결론 (Conclusion)**: 마무리 요약

**스타일**:
- 카드 형태의 흰색 배경
- 각 섹션 구분선 또는 여백으로 구분

---

### 2.8 타임라인 섹션 (TimelineSection)
**목적**: 자막 타임라인 + 해당 시점으로 이동

**UI 요소**:
- 섹션 제목: "타임라인"
- 스크롤 가능한 타임라인 목록
- 각 항목:
  - 타임스탬프 (클릭 가능, 링크 스타일)
  - 해당 시점의 자막 텍스트

**동작**:
- 타임스탬프 클릭 시 YouTube 영상의 해당 시점으로 새 탭에서 열기
  (예: https://youtube.com/watch?v=xxx&t=120)

---

### 2.9 관련 동영상 섹션 (RelatedVideosSection)
**목적**: 현재 영상과 관련된 다른 영상 추천

**UI 요소**:
- 섹션 제목: "관련 동영상"
- 최대 3개의 관련 영상 카드
- 각 카드:
  - 썸네일 이미지
  - 영상 제목
  - 채널명

**동작**:
- 카드 클릭 시 해당 YouTube 영상으로 이동 (새 탭)

---

### 2.10 푸터 (Footer)
- 간단한 앱 설명: "YouTube Summary Agent - AI 기반 영상 요약 도구"
- 회색 텍스트, 중앙 정렬

## 반응형 고려사항

- 모바일: 단일 컬럼 레이아웃
- 태블릿: 썸네일과 정보 수직 배치

---

## 사용자 플로우

1. **URL 입력** → 사용자가 YouTube URL 붙여넣기
2. **미리보기 (선택)** → 영상 정보 확인
3. **요약 시작** → 로딩 화면 (실시간 경과 시간 표시)
4. **결과 확인**:
   - 완료 배너 + 소요 시간
   - 영상 정보 카드 + Notion 링크
   - 구조화된 요약
   - 타임라인 (클릭하면 해당 시점으로 이동)
   - 관련 동영상 추천
5. **히스토리** → 이전에 요약한 영상 다시 보기

---

## 디자인 개선 포인트 (제안)

1. **다크 모드 지원**: 토글 스위치로 다크/라이트 모드 전환
2. **애니메이션**: 컴포넌트 등장 시 부드러운 fade-in/slide-up
3. **프로그레스 바**: 로딩 시 단계별 진행 상황 표시
4. **토스트 알림**: URL 복사 성공 등 피드백
5. **스켈레톤 UI**: 데이터 로딩 중 플레이스홀더 표시
